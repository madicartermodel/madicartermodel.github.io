2. Erhobene Daten

Unsere App verwendet die Meta / Facebook / Instagram Schnittstellen (Graph API). Dabei können folgende Daten verarbeitet werden:

Wir speichern diese Daten nicht dauerhaft. Sie werden nur für die Bereitstellung der App-Funktionen genutzt.

3. Zweck der Verarbeitung

Die Daten werden ausschließlich genutzt, um Automatisierungen und Integrationen (z. B. Kommentarmanagement, Benachrichtigungen) bereitzustellen.

4. Weitergabe an Dritte

5. Speicherdauer

Wir speichern keine personenbezogenen Daten dauerhaft. Alle Daten werden nur für die Dauer der aktiven Nutzung in n8n-Workflows verarbeitet und danach verworfen.
